# 生图

这里整理 UseGoodAI 相关的图片生成说明。

## 页面怎么选

| 你要做什么 | 建议看 |
| --- | --- |
| 用画布整理提示词、参考图和多轮图片结果 | [无限画布](/images/infinite-canvas) |
| 自己写脚本或服务端接口调用图片生成 | [图片生成 API](/images/image-generation) |
| 想让 Codex 帮你整理提示词、写脚本、批量生成和管理结果 | [Codex 生图高级技巧](/images/codex-image) |
| 想用 Codex App 内置生图，或分清直连 API、MCP、插件的差别 | [Codex 直接生图配置说明](/images/codex-image-direct) |

如果只是想最快生成一张图，优先看 [无限画布](/images/infinite-canvas)。如果要在自己的程序里接入图片生成接口，再看 [图片生成 API](/images/image-generation)。
