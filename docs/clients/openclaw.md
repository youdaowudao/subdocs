## 本页要说明什么

- OpenClaw 是否支持 OpenAI-compatible 接口
- 客户端里 Base URL、API Key、模型名分别填在哪里
- 哪些模型适合写代码、长上下文和普通问答
- 接入失败时应该先看客户端错误还是后台日志

## 基础信息

| 项目 | 内容 |
| --- | --- |
| API 地址 | `https://api.usegoodai.com/v1` |
| API Key | 在 UseGoodAI 后台创建 |
| 模型 | 以后台可用模型为准 |
| 重点排查 | 接口类型、模型名、分组权限 |
